# TypeScript: Fundamentos

Este proyecto está diseñado para explorar y aprender los conceptos fundamentales de TypeScript, cubriendo desde la configuración básica hasta la integración con APIs y el manejo de tipos avanzados.

## 🚀 Estructura del Proyecto

El repositorio se divide en las siguientes secciones principales:

- **Raíz**: Contiene la configuración principal de TypeScript y el código fuente de los ejercicios fundamentales.
- **servidor-api/**: Un servidor de ejemplo (API) utilizado para practicar peticiones HTTP y tipado de datos externos.

## 🛠️ Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (versión LTS recomendada)
- [npm](https://www.npmjs.com/) o [yarn](https://yarnpkg.com/)

## 📦 Instalación

1. Clona el repositorio.
2. Instala las dependencias en la raíz del proyecto:
   ```bash
   npm install
   ```
3. Si planeas usar el servidor API, navega a su carpeta e instala sus dependencias:
   ```bash
   cd servidor-api
   npm install
   ```

## ⚙️ Scripts Disponibles

- `npm run build`: Compila el código TypeScript a JavaScript.
- `npm run start`: Ejecuta el proyecto compilado.
- `npm run dev`: Ejecuta el proyecto en modo desarrollo con recarga automática.

## 📚 Temas Cubiertos

- Tipos básicos y complejos.
- Interfaces y Types.
- Configuración de `tsconfig.json`.
- Consumo de APIs con TypeScript.
